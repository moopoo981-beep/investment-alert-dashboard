# Investment Alert Dashboard

ไฟล์ชุดนี้เป็นหน้าเว็บ Dashboard สำหรับ GitHub Pages

## วิธีใช้

1. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub repo
2. เปิด Settings > Pages
3. เลือก Deploy from a branch
4. Branch: main
5. Folder: /root
6. กด Save

## แก้พอร์ต

แก้ไฟล์นี้:

data/portfolio.json

## แก้ผลวิเคราะห์ข่าว

ตอนนี้ใช้ไฟล์ตัวอย่าง:

data/results.json

ขั้นต่อไปค่อยเพิ่ม GitHub Actions ให้ดึงข่าวและใช้ Gemini วิเคราะห์อัตโนมัติ
